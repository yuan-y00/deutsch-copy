如何添加到手机主屏幕

iPhone / iPad（Safari）：

  1. 用 Safari 打开 GitHub Pages 地址。
  2. 点击底部中间 分享按钮（↑）。
  3. 选择 「添加到主屏幕」。
  4. 点击右上角「添加」。
  5. 从主屏幕打开 Deutsch Copy 图标。

Android（Chrome）：

  1. 用 Chrome 打开 GitHub Pages 地址。
  2. 点击右上角 菜单（⋮）。
  3. 选择 「添加到主屏幕」或「安装应用」。

注意：所有音频播放需点击按钮触发（iPhone Safari 限制）。
首次部署后建议点击「部署检查」验证资源路径。
